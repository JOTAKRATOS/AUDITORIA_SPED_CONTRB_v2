SPED Contribuições Project

Um projeto Python para converter arquivos SPED em Excel automaticamente.

Descrição do Projeto

Este projeto lê um arquivo de SPED Contribuições e converte os dados em planilhas Excel, facilitando a análise e organização das informações fiscais.

O script oferece uma interface simples para que o usuário selecione o arquivo SPED, processando e gerando um arquivo Excel com todos os registros organizados por abas.
